Conclusion

Pros

  • It's open source!
  • It's very productive
  • Sharable code
  • Can be added to existing native app
  • Can be extended with components written in native code
  • Easy to debug (using Chrome dev tools)
  • Web-like approach
  • Works with Redux  
  • Opens whole new world for JS developers

Conclusion

Cons

  • It's still young
  • Ecosystem isn't big
  • You still have to be familiar with key concepts of native app development
  • Bridge performance is low
  • Hard to implement some trivial for Web things
